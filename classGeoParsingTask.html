Detailed Description

Tâche asynchrone encapsulant l'exécution du GeoParser.

Cette classe est purement statique : elle n'a pas d'état et ne doit pas être instanciée.

Usage :

GeoParsingTask::launch(hWnd, "C:/data/reseau.geojson");

void GeoParsingTask::launch ( HWND  hWnd,
const std::string &  geoJsonPath 
)
static

Lance le parsing GeoJSON dans un thread séparé et détaché.

Le thread émet les messages suivants vers hWnd :

  • WM_PROGRESS_UPDATE (WPARAM = avancement 0–100) à chaque étape.
  • WM_PARSING_SUCCESS à la fin d'un parsing réussi.
  • WM_PARSING_ERROR (LPARAM = pointeur std::string) en cas d'échec.
Parameters
hWndFenêtre destinataire des messages de progression. Doit rester valide pendant toute la durée du parsing.
geoJsonPathChemin absolu vers le fichier GeoJSON à analyser.
Warning
Le std::string* transporté par WM_PARSING_ERROR doit être libéré (delete) par le destinataire du message.
