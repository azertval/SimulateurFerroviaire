Description détaillée

Gestion du cycle de vie de l'application Win32.

La classe Application encapsule les trois étapes fondamentales du démarrage d'une application Win32 :

  1. Enregistrement de la classe de fenêtre (RegisterClassExW).
  2. Création et affichage de la fenêtre principale (MainWindow).
  3. Exécution de la boucle de messages (GetMessage / DispatchMessage).
Note
Une seule instance de cette classe doit exister par processus.
