Description détaillée

Contexte BFS d'un nœud en file d'attente.

Le champ arrivedViaDeviation est le mécanisme central du layout : il indique si le nœud a été enqueué via une arête DEVIATION. Quand un PCCSwitchNode est dépilé avec ce flag à true, sa voie ROOT (forward) est positionnée à x-1 (amont) plutôt qu'à x+1 (aval).
