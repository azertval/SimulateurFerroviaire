Detailed Description

Déclaration du sélecteur de sauvegarde de fichier GeoJSON.

La classe FileSaveDialog encapsule l'appel à GetSaveFileNameA derrière une interface moderne retournant un std::optional. Cela évite de mélanger la logique de sélection de fichier avec le gestionnaire de commandes de MainWindow.
