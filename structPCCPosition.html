Detailed Description

Position logique d'un nœud dans le schéma PCC.

Calculée par PCCLayout, indépendante des coordonnées GPS.

  • x : profondeur BFS depuis le terminus le plus à l'ouest.
  • y : rang vertical — 0 = backbone, +1 = branche haute, -1 = branche basse.
Note
Initialisée à {0, 0} par défaut — non significative tant que PCCLayout n'a pas été exécuté.
