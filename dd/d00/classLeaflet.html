Description détaillée

Utilitaire statique de génération du code HTML Leaflet.

Produit une page HTML autonome intégrant la bibliothèque Leaflet, une carte centrée sur Paris et une API JavaScript exposée à C++.

Usage :

m_webViewPanel.navigateToString(Leaflet::leafletHtml());
