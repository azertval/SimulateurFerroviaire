class  Phase2_GeometricIntersector
 

Description détaillée

Phase 2 du pipeline — calcul des intersections géométriques.

Responsabilité unique : pour chaque paire de segments du RawNetwork, détecter les intersections géométriques et les stocker dans IntersectionData.

Utilise un spatial grid binning pour limiter les tests aux segments candidats — complexité O(n·k) au lieu de O(n²).

Ce que Phase2 ne fait PAS
  • Elle ne découpe pas les segments (Phase3_NetworkSplitter).
  • Elle ne snap pas les nœuds (Phase4_TopologyBuilder).
  • Elle ne classe pas les nœuds (Phase5_SwitchClassifier).
Note
Classe entièrement statique — instanciation interdite.
