Detailed Description

Bloc de voie droite produit par le parseur GeoJSON.

Cycle de vie : Phase 4 (extraction) — créé avec coordonnées WGS-84 et longueur calculée. Phase 5a (découpe) — peut être subdivisé ; les morceaux héritent du suffixe _cN. Phase 5b (câblage) — neighbours peuplé avec les IDs adjacents.
