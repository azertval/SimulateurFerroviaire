# Simulateur Ferroviaire

## Description

Reconstruction et visualisation d’un réseau ferroviaire depuis GeoJSON.

---

## Pipeline

GeoJSON → GeoParser → Graph → Switch → Map

---

## Modules

- GeoParser
- TopologyGraph
- SwitchBlock
- Map (WebView2 + Leaflet)

---

## Objectif

Simulateur ferroviaire interactif
