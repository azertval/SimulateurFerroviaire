void TCORenderer::draw ( HDC  hdc,
const RECT &  rect,
Logger &  logger 
)
static

Point d'entrée unique du rendu TCO.

Lit TopologyRepository, calcule la projection GPS → pixels, remplit le fond noir, puis délègue aux méthodes privées de rendu des StraightBlocks et SwitchBlocks. Aucun état n'est conservé entre deux appels.

Si TopologyRepository est vide (parsing non encore effectué), seul le fond noir est dessiné — aucune exception n'est levée.

Parameters
hdcContexte de périphérique GDI valide, fourni par BeginPaint.
rectZone cliente dans laquelle dessiner (origine = (0,0)).
loggerRéférence au logger HMI, fourni par PCCPanel.

◆ drawSwitches()

void TCORenderer::drawSwitches ( HDC  hdc,
const Projection &  proj,
Logger &  logger 
)
staticprivate

Dessine tous les SwitchBlocks (jonction + 3 branches).

Pour chaque switch orienté (isOriented() == true), dessine :

  • tip root → jonction (couleur état switch)
  • jonction → tip normal (vert si branche active, sinon couleur état)
  • jonction → tip déviation (jaune si branche active, sinon couleur état)
  • disque de 4 px à la jonction

Les switches non orientés (parsing incomplet) sont ignorés silencieusement.

Parameters
hdcContexte de périphérique GDI.
projProjection GPS → pixels courante.
loggerRéférence au logger HMI.

Projette un point GPS en coordonnées pixels.

Applique une normalisation linéaire avec marge sur [0, width] × [0, height]. L'axe Y est inversé : latitude maximale → Y = 0.
