Description détaillée

Représentation d'une coordonnée plane en mètres (système métrique UTM).

Utilisé en interne par le pipeline pour tous les calculs métriques (construction du graphe, accrochage sur grille, interpolation, angles). Les coordonnées publiques (sorties du parseur) sont toujours en LatLon WGS-84.

Exemple :

CoordinateXY metricPoint{ 652345.0, 5862234.0 }; // UTM zone 31N
double x = metricPoint.x;
double y = metricPoint.y;
