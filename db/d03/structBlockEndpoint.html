Description détaillée

Extrémité d'un bloc — nœud frontière + ID du bloc voisin.

frontierNodeId vaut SIZE_MAX pour les endpoints internes des sous-blocs produits par subdivision (maxSegmentLength) : il n'existe aucun nœud topologique réel à ces positions intermédiaires. Phase8_RepositoryTransfer::resolveStraight teste cette valeur pour éviter d'écraser les pointeurs de chaîne posés par Phase6_BlockExtractor::registerStraight.
