Detailed Description

Phases 3, 4, 5a et 5b — extraction de la topologie ferroviaire depuis le graphe.

Phase 3 (detectSwitches) : Identifie les nœuds de jonction (degré ≥ 3) et crée les objets SwitchBlock.

Phase 4 (extractStraights) : Marche le graphe depuis chaque nœud frontière. Collecte les chemins droits maximaux entre nœuds frontières. Déduplique les chemins par empreinte géométrique.

Phase 5a (splitLongStraights) : Subdivise tout StraightBlock dépassant maxStraightLengthMeters. Les morceaux sont nommés "s/N_c1", "s/N_c2", etc.

Phase 5b (wireTopology) : Peuple SwitchBlock::branchIds avec les IDs des StraightBlocks adjacents. Peuple StraightBlock::neighbourIds avec les IDs des voisins (Switch ou Straight).
